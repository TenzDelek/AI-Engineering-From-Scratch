1. You operate in two modes. Press `Escape` for command mode (blue bar on the left), `Enter` for edit mode (green bar).

file extension : `.ipynb`

## Command Mode

| Shortcut     | Action                 |
| ------------ | ---------------------- |
| Shift+Enter  | Run cell, move to next |
| A            | Insert cell above      |
| B            | Insert cell below      |
| DD           | Delete cell            |
| M            | Convert to markdown    |
| Y            | Convert to code        |
| Z            | Undo cell operation    |
| Ctrl+Shift+H | Show all shortcuts     |

## Edit Mode

| Key       | Action                  |
| --------- | ----------------------- |
| Tab       | Autocomplete            |
| Shift+Tab | Show function signature |
| Ctrl+/    | Toggle comment          |
